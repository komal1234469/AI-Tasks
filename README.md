# 🚀 NLP Text Preprocessing & Word Frequency Analysis

## 📌 Overview
This project demonstrates a complete **Natural Language Processing (NLP) preprocessing pipeline** along with **word frequency visualization**.

It takes user input text, cleans and processes it, and then displays the most frequent words using a bar chart.

---

## 🧠 Features
- ✅ Text Cleaning (lowercase, number removal)
- ✅ Tokenization
- ✅ Stopword Removal
- ✅ Lemmatization
- ✅ Word Frequency Analysis
- ✅ Data Visualization using Matplotlib

---

## ⚙️ Tech Stack
- Python 🐍
- NLTK
- NumPy
- Matplotlib

---

## 📂 Project Workflow

1. **Input Text**
   - User enters raw text

2. **Preprocessing**
   - Remove numbers
   - Convert to lowercase
   - Tokenize words
   - Remove stopwords & punctuation
   - Apply lemmatization

3. **Analysis**
   - Count word frequencies using `Counter`

4. **Visualization**
   - Plot top frequent words using bar chart
